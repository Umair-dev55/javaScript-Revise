JavaScript Functions with Parameters

In this example, we demonstrate how to pass parameters to JavaScript functions and handle them properly.

Code Example

function parameter(x) {
    console.log(x);
}
parameter("Umair");

function second(y) {
    console.log(y);
}
second("Fahad");

function third(str) {
    console.log(str);
}
third("ASAD");

function sum(str) {
    const parts = str.split(',');  // Split the string by comma
    console.log(parts[0] + parts[1]);  // Combine the two parts
}
sum('Add 1st , Add 2nd');

Output

Umair
Fahad
ASAD
Add 1st Add 2nd
