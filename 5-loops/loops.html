Understanding Loops in JavaScript

General Loops:

  • For Loop
  • While Loop
  • Do-While Loop
  • For-Of Loop
  • For-In Loop

For Loop Example:

// For Loop Example for (let x = 1; x <= 10; x++) { console.log("Value of x is :", x); } for (let y = 50; y <= 65; y++) { console.log("The Value of Y is :", y); }

While Loop Example:

let j = 11; while (j <= 11) { console.log("Umair Ali"); j++; }

Do-While Loop Example:

let w = 20; do { console.log("Apna College"); w++; } while (w <= 26);

For-Of Loop (String) Example:

let str = "SUBHANI"; let length = 0; for (let val of str) { console.log("value of val is :", val); length++; } console.log("length of str is : ", length);

For-In Loop (Object) Example:

let store = { productId: 123, productName: "Laptop", productCategory: "Computers", productQuantity: 56, }; for (let key in store) { console.log("key = ", key, "value =", store[key]); }

For-Of Loop Example (String):

let stringg = "Umair"; let sizee = 0; for (const element of stringg) { console.log(element); sizee++; } console.log(sizee);

For-In Loop Example (Products):

let products = { computer: 2, laptop: 4, lcd: 6, monitor: 10, bluetooth: 15, handfree: 150, }; for (const key in products) { console.log("key is : ", key, "and its value is :", products[key]); }
